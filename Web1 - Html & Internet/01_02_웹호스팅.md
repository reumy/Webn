# 웹호스팅
- 웹서버를 대행해주는 서비스

## [GitHub](https://github.com) pages
- 정적(static) 웹사이트 호스팅
### 1. New repository : 저장소(repository) 생성 (코드 보관소)
### 2. upload files : 프로젝트 파일 업로드
### 3. Settings
- GitHub Pages 에서 master branch 선택
- Select branch 에서 master branch 선택 후 save
- 주소가 표시됨
### 4. 주소로 접속하면 업로드한 파일의 웹페이지가 출력됨
- 안나오면 5분 후 다시 접속
