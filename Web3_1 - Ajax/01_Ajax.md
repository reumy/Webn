# Ajax
- 부분적으로 웹페이지를 갱신하도록 도와주는 기술
- 웹브라우저가 ajax를 통해서 웹서버와 통신함
- 장점 : 변경된 부분에만 집중할 수 있는 사용성의 증대와 필요한 부분만 로드해 서버와 사용자 입장에서도 시간, 돈, 네트워크 자원이 절약됨
- 개발자도구 Network : 브라우저와 웹서버가 서로 어떠한 정보를 주고받는지(통신)를 모니터링 해줌
