# CSS
- 해당하는 정보의 이름을 정확히 알아야 스스로 원하는 정보를 

알아낼 수 있음
```
a { color : red };
```
- selector : 선택자 ( a )
- declaration : 선언, 효과 ( color : red )
- property : 속성 ( color )
- value : 값 ( red )

## 박스모델
#### block
- block과 inline은 display 속성의 기본값이며, 변경이 가능함
- display : block, inline, none
- block level element : 화면 전체를 씀
- inline element : 자기 자신의 컨텐츠 만큼만 씀
